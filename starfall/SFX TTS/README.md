# SFX TTS

Script that allows you to use TTS voices and play audio files via chat messages. Supports Oddcast, DECTalk and Tik Tok voices.

`sf_filedata`
```
sf_filedata
└───sfx_tts
    ├───aliases.json
    └───samples
        └───. . .
```

`aliases.json`
```json
{
  "file_name": [
    "alias text/trigger word",
    . . .
  ],
  . . .
}
```
